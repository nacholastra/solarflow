# Go-to-market — validación España (90 días)

Este kit implementa el plan de valoración de mercado: **no pivotar producto ni país** hasta demostrar pagos reales en España.

## Objetivo

| Métrica | Meta (día 90) |
|---------|----------------|
| Conversaciones cualificadas | ≥ 50 |
| Demos realizadas | ≥ 10 |
| Clientes de pago (Basic o Pro) | ≥ 3 |
| Widget vivo ≥ 60 días | ≥ 1 (ideal ≥ 2) |

## Regla de oro

La landing **no vende sola**. El cierre es humano (llamada / WhatsApp / demo). Usa el trial de 14 días y el early bird (−30%, primeros 20) como palanca, no como sustituto del outreach.
